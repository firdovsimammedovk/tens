TT-Forge-ONNX
=============

**TT-Forge-ONNX** is a graph compiler that optimizes and transforms computational graphs for deep learning
models on Tenstorrent hardware. It provides ONNX, TensorFlow, and PaddlePaddle frontend abstraction and
lowers models through the `TT-MLIR <https://docs.tenstorrent.com/tt-mlir/>`_ backend.
