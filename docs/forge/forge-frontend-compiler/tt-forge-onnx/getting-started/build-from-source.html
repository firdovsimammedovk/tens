Build from Source

This guide describes how to build TT-Forge-ONNX locally. Use this method if you want to develop for TT-Forge-ONNX.

Note

TT-Forge-ONNX is a framework-agnostic frontend that converts any model to a generic Intermediate Representation (IR), which is then converted to a Tenstorrent-specific IR for use with Tenstorrent hardware. TT-Forge-ONNX is for use with single-chip systems only.

Configuring Your Hardware

If you already configured your hardware, skip this section.

  1. Configure your hardware with TT-Installer using the Quick Installation guide.

  2. Reboot your machine.

  3. Activate the virtual environment:

    source ~/.tenstorrent-venv/bin/activate

  4. Verify hardware detection:

    tt-smi

Prerequisites

Dependency

Version

Clang

17

Ninja

latest

CMake

latest

Python

3.12

On Ubuntu 22.04, install dependencies as follows:

sudo apt update -y
sudo apt upgrade -y

Installing Clang 17

wget https://apt.llvm.org/llvm.sh
chmod u+x llvm.sh
sudo ./llvm.sh 17
sudo apt install -y libc++-17-dev libc++abi-17-dev
sudo ln -s /usr/bin/clang-17 /usr/bin/clang
sudo ln -s /usr/bin/clang++-17 /usr/bin/clang++

Verify the installation:

clang --version
clang++ --version

If you need to switch between Clang versions:

sudo update-alternatives --install /usr/bin/clang clang /usr/bin/clang-17 100
sudo update-alternatives --install /usr/bin/clang++ clang++ /usr/bin/clang++-17 100

Installing Ninja

sudo apt install ninja-build

Installing CMake

sudo apt install cmake

Verify:

cmake --version

Installing Python 3.12

python3 --version  # check existing version

If Python 3.12 is not installed:

sudo apt install python3.12

Building the Environment

  1. Clone the repository:

    git clone https://github.com/tenstorrent/tt-forge-onnx.git
cd tt-forge-onnx

  2. Build the project:

    cmake -B build -G Ninja
cmake --build build

Building the Docs

TT-Forge-ONNX uses mdbook for documentation. Install it:

sudo apt install cargo
cargo install mdbook

Then build the docs:

mdbook build docs

Build Cleanup

To clean the build directory:

cmake --build build --target clean

Or remove the build directory entirely:

rm -rf build

Useful Build Environment Variables

Variable

Description

FORGE_EXTRACT_UNIQUE_OP_CONFIG_AT

Extract unique op configurations at a specific compilation stage.

FORGE_PRINT_UNIQUE_OP_CONFIG

Set to 1 to print all unique op configs to console.

FORGE_EXPORT_UNIQUE_OP_CONFIG_FILE_TYPE

Set to csv or xlsx to export op configs.

FORGE_EXPORT_UNIQUE_OP_CONFIG_DIR_PATH

Directory path for exported op config files.

FORGE_EXPORT_UNIQUE_OP_CONFIG_CSV_DELIMITER

Delimiter for CSV export (default: /).

Note

If you encounter issues, request assistance on the TT-Forge-ONNX Issues page.
