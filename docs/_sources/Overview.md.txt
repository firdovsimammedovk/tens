# Overview

Welcome to Tenstorrent - a high-performance computing platform designed for modern AI workloads, from model development to large-scale deployment.

Tenstorrent provides a vertically integrated stack that combines specialized hardware, low-level programming interfaces, and high-level APIs to enable efficient execution of machine learning models.

---

## What You Can Do

With Tenstorrent, you can:

- Run and deploy machine learning models on Tenstorrent hardware
- Optimize performance for inference and training workloads
- Develop custom kernels and operations
- Scale workloads across multiple devices and systems
- Analyze and debug execution using built-in tools

---

## Platform Overview

Tenstorrent consists of four main layers:

### Hardware

Custom AI accelerators and systems designed for high-throughput computation.

- Tensix cores
- High-bandwidth memory architecture
- Network-on-Chip (NoC)
- Multi-chip systems

Learn more: [Hardware Overview](aibs/index.rst)

---

### Software Stack

A layered software platform that provides different levels of abstraction:

- **TT-NN** - high-level API for model development
- **TT-Metalium** - low-level programming model for kernels and execution
- **Compiler / Runtime** - execution and optimization layer

Learn more: [Software Overview](software/Overview.md)

---

### Tools

Utilities for monitoring, debugging, and optimizing workloads:

- `tt-smi` - system monitoring
- Profiler - performance analysis
- Topology Explorer - device layout
- Visualizer - execution insights

Learn more: [Tools](tools/index.rst)

Prebuilt examples and workflows to help you get started quickly:

- Multi-device execution
