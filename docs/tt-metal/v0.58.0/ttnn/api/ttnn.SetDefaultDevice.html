ttnn.SetDefaultDevice

ttnn.SetDefaultDevice(device: MeshDevice)

Set the process default device.

Signature 

ttnn.SetDefaultDevice(device: ttnn.device.Device) -> None

Description

Updates process-level default device state used by APIs that support implicit device context.

Parameters

  • device: ttnn.device.Device - Device to set as default.

Returns

  • None - Default device state is updated.

Raises

  • RuntimeError - If the provided device is invalid.

Notes

  • Affects only the current process.

Examples 

>>> import ttnn
>>> device = ttnn.open_device(device_id=0)
>>> ttnn.SetDefaultDevice(device)

See Also

  • ttnn.GetDefaultDevice - Query current default device.

  • ttnn.open_device - Open a device handle.

  • ttnn.close_device - Close device handles.
