TT-Forge Overview

TT-Forge is the end-to-end stack. TT-Forge-ONNX is the frontend that accepts ONNX, TensorFlow, and PaddlePaddle models and converts them into a generic IR.

Framework Model (ONNX / TF / PaddlePaddle)
           │
           ▼
      TT-TVM (Relay IR)
           │
           ▼
    TT-Forge-ONNX (Forge IR)
           │
           ▼
        TT-MLIR
           │
           ▼
    Device Binary (Flatbuffer)
