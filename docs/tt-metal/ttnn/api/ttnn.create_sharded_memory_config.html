ttnn.create_sharded_memory_config

ttnn.create_sharded_memory_config(shape: Shape | Tuple[int, ...] | List[int], core_grid: CoreGrid | CoreRangeSet, strategy: ShardStrategy, orientation: ShardOrientation | None = None, use_height_and_width_as_shard_shape: bool = False) → MemoryConfig

Create a sharded ttnn.MemoryConfig for sharded tensor operations.

Signature 

ttnn.create_sharded_memory_config(
    shape: ttnn.Shape | tuple[int, ...] | list[int],
    core_grid: ttnn.CoreGrid | ttnn.CoreRangeSet,
    strategy: ttnn.ShardStrategy,
    orientation: ttnn.ShardOrientation | None = None,
    use_height_and_width_as_shard_shape: bool = False,
) -> ttnn.MemoryConfig

Description

Builds a MemoryConfig with a ShardSpec describing how tensor data is partitioned across compute cores in L1.

The helper validates input shape, converts core_grid into a ttnn.CoreRangeSet, computes shard shape according to the selected sharding strategy, and returns a ready-to-use configuration object.

Parameters

  • shape: ttnn.Shape | tuple[int, …] | list[int] - Logical tensor shape.

  • core_grid: ttnn.CoreGrid | ttnn.CoreRangeSet - Core topology used for sharding.

  • strategy: ttnn.ShardStrategy - One of HEIGHT, WIDTH, or BLOCK.

  • orientation: ttnn.ShardOrientation | None - Core traversal order; defaults to ROW_MAJOR when None.

  • use_height_and_width_as_shard_shape: bool - If True, uses (height, width) directly as shard shape.

Returns

  • ttnn.MemoryConfig - Memory configuration with L1 buffer type and computed shard spec.

Raises

  • RuntimeError - If shape or core_grid has an unsupported type or invalid structure.

  • RuntimeError - If strategy or orientation is invalid.

  • RuntimeError - If shard geometry is not divisible by the selected core topology.

  • RuntimeError - If core_grid is a ttnn.CoreRangeSet while use_height_and_width_as_shard_shape is False.

Notes

  • This helper creates L1 sharded configs only.

  • When use_height_and_width_as_shard_shape is False, shape is interpreted as [prod(batch_dims), height, width] for shard computation.

  • For advanced/non-divisible sharding flows, see ttnn.create_sharded_memory_config_().

Examples

Block sharding over a rectangular grid:

>>> import ttnn
>>> grid = ttnn.CoreGrid(y=8, x=8)
>>> mem_cfg = ttnn.create_sharded_memory_config(
...     shape=(1, 1, 1024, 1024),
...     core_grid=grid,
...     strategy=ttnn.ShardStrategy.BLOCK,
...     orientation=ttnn.ShardOrientation.ROW_MAJOR,
... )

Using explicit shard shape from tensor height/width:

>>> import ttnn
>>> grid = ttnn.CoreGrid(y=4, x=4)
>>> mem_cfg = ttnn.create_sharded_memory_config(
...     shape=(1, 1, 256, 256),
...     core_grid=grid,
...     strategy=ttnn.ShardStrategy.HEIGHT,
...     use_height_and_width_as_shard_shape=True,
... )

See Also

  • ttnn.create_sharded_memory_config_() - Extended helper with additional shard-shape handling.

  • ttnn.ShardSpec - Describes shard grid, shape, and orientation.

  • ttnn.MemoryConfig - Container for tensor memory layout and placement metadata.
