TT-MLIR
=======

TT-MLIR is the backend compiler for TT-Forge. It takes a Tenstorrent-specific graph intermediate
representation (IR) that was created by one of the TT-Forge frontends (TT-Forge-FE or
TT-XLA) and lowers it into Tenstorrent hardware-specific code using the MLIR framework.

TT-MLIR leverages the Multi-Level Intermediate Representation (MLIR) infrastructure to provide a
flexible, extensible compiler backend that can target various Tenstorrent hardware architectures
including Grayskull, Wormhole, and Blackhole.

Key Features
------------

* **MLIR-Based Infrastructure**: Built on top of the MLIR framework for modularity and extensibility
* **Multi-Target Support**: Targets Grayskull, Wormhole, and Blackhole architectures
* **Optimized Compilation**: Advanced optimization passes for Tenstorrent hardware
* **Flexible IR**: Multiple dialect levels (TTIR, TTNN, TTMetal) for progressive lowering
* **Runtime Integration**: Seamless integration with TTRT (Tenstorrent Runtime)

Architecture Overview
---------------------

TT-MLIR employs a multi-stage compilation pipeline:

1. **TTIR (TT Intermediate Representation)**: High-level operations that are target-independent
2. **TTNN (TT Neural Network)**: Mid-level operations optimized for neural network workloads
3. **TTMetal**: Low-level operations that map directly to Tenstorrent hardware primitives

Each stage includes optimization passes specific to that abstraction level.
