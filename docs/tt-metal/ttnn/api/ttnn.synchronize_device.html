ttnn.synchronize_device

ttnn.synchronize_device(device: MeshDevice)

Block until pending operations on a device are completed.

Signature 

ttnn.synchronize_device(device: ttnn.device.Device) -> None

Description

Synchronizes command queues for the provided device so queued operations complete before control returns to the caller.

Parameters

  • device: ttnn.device.Device - Device handle to synchronize.

Returns

  • None - Synchronization completes with no return value.

Raises

  • RuntimeError - If synchronization fails or times out.

Notes

  • Useful before teardown and when benchmarking to establish clear timing boundaries.

Examples 

>>> import ttnn
>>> device = ttnn.open_device(device_id=0)
>>> ttnn.synchronize_device(device)
>>> ttnn.close_device(device)

See Also

  • ttnn.open_device - Open device handles for work submission.

  • ttnn.close_device - Close a device after synchronization.

  • ttnn.manage_device - Automatic lifecycle wrapper.
